# Semantic Similarity-Based Plagiarism Checker for Academic Content

## Overview

An AI-powered plagiarism detection system that identifies semantic and paraphrased plagiarism in academic documents using Sentence-BERT (SBERT) embeddings and cosine similarity.

## Objectives

* Detect semantic plagiarism by analyzing the contextual meaning of text.
* Compare academic documents using Sentence-BERT and cosine similarity.
* Generate plagiarism reports with similarity scores and risk assessment.

## Features

* PDF and DOCX support
* Semantic similarity detection
* Sentence-level comparison
* Similarity score generation
* Risk assessment
* Teacher-focused workflow
